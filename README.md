# CertFlow - Learning Path Tracker

CertFlow is a beautifully designed, lightweight Single Page Application (SPA) built to help you track your learning journeys, courses, skills, and certifications all in one place. All data is securely stored locally in your browser, meaning it's blazing fast and completely private!

## ✨ Features

- **Dashboard**: Get a bird's-eye view of your learning progress with a daily motivation widget, real-time stats, and a skill spotlight.
- **Courses**: Track courses by status (To Start, In Progress, Completed), add progress percentages, and link them to specific providers and skills.
- **Providers & Categories**: Keep your learning organized by tagging courses with specific providers (e.g., Coursera, Udemy) and grouping skills by customized categories.
- **Pre-Loaded Skills**: Comes with 96 carefully categorized, industry-standard skills out of the box so you can hit the ground running.
- **Profile**: Customize your developer portfolio by adding your name, role, bio, contact details, and links to your GitHub, LinkedIn, and personal website. Also supports local profile picture uploads!
- **Manage Hub**: A centralized, intuitive interface to easily modify, delete, or add new items to your learning tracker.

## 🚀 Getting Started

Since CertFlow requires no backend server, getting started is incredibly simple:

1. Clone or download this repository.
2. Open `index.html` in any modern web browser.
3. Start tracking your learning!

## 🛠️ Technology Stack

- **HTML5 & CSS3**: Pure, vanilla web technologies.
- **Vanilla JavaScript**: Zero dependencies, framework-free implementation.
- **Phosphor Icons**: Beautiful, consistent iconography.
- **Local Storage**: All data is persisted directly in your browser's local storage for maximum privacy and zero latency.
